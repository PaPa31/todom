Make bootable USB (ebox2300)

source: http://www.dmp.com.tw/tech/vortex86sx/

Installing FreeDOS

  • Download the latest version of Rufus from the rufus website

I downloaded this: rufus-4.4_x86.exe

  • Launch Rufus (I ran it from Windows 11)
  • Select "FreeDOS" in the menu
  • Click "Format"

Download DEMO Images

  • Download Demo files (2007-05-29) (29.3 MB)
  • Unzip this ZIP file
  • Copy/paste to USB drive (if duplicate file names are found, replace them with the extracted ones)

You can now connect this USB drive to your ebox2300 and boot from it.
